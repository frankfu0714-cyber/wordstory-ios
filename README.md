# wordstory (iOS)

A SwiftUI iOS companion to the [wordstory web app](https://wordstory.vercel.app). Save unfamiliar words, then let Gemini weave them into short stories, articles, dialogues, letters, or poems — context-based vocab learning beats rote.

## Stack
- **UI:** SwiftUI (iOS 17+)
- **Persistence:** SwiftData (`@Model Word`)
- **Networking:** `URLSession` + `Codable` — calls the same `https://wordstory.vercel.app/api/{define,generate}` endpoints
- **Localization:** Xcode String Catalog (`Localizable.xcstrings`) — English + 繁體中文
- **No third-party packages.** Pure Apple stack.

## Open in Xcode

1. Install xcodegen if needed: `brew install xcodegen`
2. Generate the Xcode project:
   ```
   cd /Users/frank/wordstory-ios
   xcodegen generate
   ```
3. Open it:
   ```
   xed wordstory.xcodeproj
   ```
4. In Xcode, wait for indexing, then press **⌘R** to run on the simulator.

Signing should auto-detect Frank's team (`632H46CT74`) since it's hard-coded in `project.yml`. If it doesn't, open the target → Signing & Capabilities → choose the team manually.

## Upload to TestFlight

1. Select **Any iOS Device (arm64)** in the run destination.
2. **Product → Archive**.
3. When the Organizer opens, choose **Distribute App → App Store Connect → Upload**.
4. Wait for processing (5–15 min). Then add internal testers in App Store Connect → TestFlight.

You may need to bump `CURRENT_PROJECT_VERSION` in `project.yml` between uploads (App Store Connect requires a new build number for each new archive).

## Notes for future work
- The placeholder icon is a 1024×1024 cream-and-burgundy tile with the character "詞" generated by Pillow. Replace with a designed icon at any time — drop the new file as `Assets.xcassets/AppIcon.appiconset/AppIcon-1024.png` (same filename, same size).
- Language toggle in Settings writes to `AppleLanguages` so it persists across launches. The app also applies `.environment(\.locale, ...)` at the root for same-session updates, but a full app relaunch is recommended after switching.
- The story-highlight tap mechanism uses an in-app `wordstory://` URL scheme intercepted by `OpenURLAction`. No `CFBundleURLTypes` registration is needed because the URL is never delivered to the system.
